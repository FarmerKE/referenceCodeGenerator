## refCodeGen Package

This is a code generator package built in R.

### Installation

Install it from github use:

```
remotes::install_github("FarmerKE/referenceCodeGenerator")
```

To use its sole function, all of which will generate a unique reference:
```
reference_code()
```
```
reference_code("SH")
```
```
reference_code("SH", 1)
```
